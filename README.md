# NLRP3 Inflammasome Drug Repurposing Pipeline

This repository contains a research-grade Machine Learning pipeline that identifies novel inhibitor candidates for the **NLRP3 Inflammasome** from a database of FDA-approved small-molecule drugs. The purpose of this framework is to computationally simulate and predict Drug/Target binding activities (`pIC50`) to repurpose existing, safe drugs for new inflammatory diseases.

## 🧬 Project Architecture
The drug repurposing process relies heavily on integrating Biology, Molecular Chemistry, and Machine Learning rather than predicting arbitrary drug-disease pairs. We map properties like 3D molecular structures to target affinity:
1. **Ligand-Based Feature Engineering**: Converts canonical `SMILES` strings into deep topological arrays using **Morgan Fingerprints** (2048-bit radius) which structurally define the bonds, rings, and atoms.
2. **Lipinski's Rule of Five**: We calculate basic pharmacokinetic parameters (LogP, MW, H-Donors/Acceptors) using RDKit to model cellular transport mechanisms. 
3. **Random Forest Regression**: Trained strictly on valid descriptors, predicting the experimental binding interaction constant (`pIC50`) between the drug and the NLRP3 protein domain.
4. **FDA Virtual Screening Engine**: Dynamically queries thousands of safe, FDA-approved drugs (e.g., from ClinTox) and simulates their bindings against the pipeline to rank the best novel inhibitors.

## 📂 Project Structure
* `data/`: Contains the cleaned, foundational ML-ready active/inactive datasets for known NLRP3 inhibitors (sourced initially via databases like ChEMBL). It also houses the RDKit featurized outputs.
* `models/`: Stores the persisted `.pkl` output of our trained Random Forest model. 
* `results/`: Contains the output predictions, specifically `top_fda_candidates.csv` generated by the Virtual Screening pipeline.
* `archive/`: Old notebooks and superseded iterations not needed for the final pipeline.

### Core Pipeline Scripts:
1. `01_data_prep.py`: Sanitizes leaky metrics and utilizes `RDKit` to extract molecular fingerprints.
2. `02_train_model.py`: Fits and validates the Regression models evaluating topological structures against real clinical affinities. 
3. `03_repurpose_fda.py`: Fetches a public subset of FDA-approved drugs, standardizes their features, runs the predictive weights, and ranks the candidates computationally.

## 🚀 How to Execute
**Prerequisites**: Python 3.10+ and a standard Data Science Toolkit.

1. **Install requirements:**
   ```bash
   pip install -r requirements.txt
   ```
2. **Initialize the Dataset Features:**
   ```bash
   python 01_data_prep.py
   ```
3. **Train the ML Model:**
   ```bash
   python 02_train_model.py
   ```
4. **Screen the FDA Database for Repurposing:**
   ```bash
   python 03_repurpose_fda.py
   ```
